# Projects

Tools I’ve scoped and built end-to-end. Each one uses AI to speed up security work and to explain results in plain English.

01 CLI · Python

Security Alert Triage CLI

A command-line tool that ingests security alerts, scores them by severity and context, and flags likely false positives. It uses Claude to summarize each alert in a couple of plain-English sentences, so analysts can prioritize the queue and decide what to escalate in seconds instead of minutes.

  • Python
  • Claude API
  • Click / argparse
  • JSON

02 Web App · Flask

Phishing Email Analyzer

Paste in a raw email and the analyzer extracts indicators of compromise — sender anomalies, suspicious URLs, attachments, and header red flags. Claude then turns those technical findings into a clear risk explanation and a recommended next step that a non-technical user can actually follow.

  • Python
  • Flask
  • Claude API
  • IOC parsing
  • regex
</> GitHub
03 Dashboard · JS

Security Ops Mini-Dashboard

A lightweight dashboard that visualizes mock alert data with severity heatmaps and trend charts. An “Explain this trend” button calls Claude to summarize what a spike or pattern likely means and what to check next — making the data readable for stakeholders, not just analysts.

  • JavaScript
  • Chart.js
  • Claude API
  • Flask (API)
